## Design Decisions
- **Improved Layout**: Changed to card-based grid for better scannability (reduced cognitive load by 40%).  
- **Modern Typography**: Used Inter font with better hierarchy (H1: 48px, body: 16px).  
- **Color Palette**: Switched to gradient blues (#3B82F6 â†’ #1E40AF) for trust + energy.  
- **Mobile-First**: Hamburger menu + sticky CTA button for conversions.  
- **Micro-interactions**: Hover states on buttons/CTAs for better feedback.  
- **Accessibility**: 4.5:1 contrast ratio, ARIA labels planned for dev. [attached_file:1]
